# Discord 브리핑 메시지 형식 가이드

## 메시지 구조 (2단)

```
🌅 오늘의 브리핑 (MM/DD 요일)

⭐ 오늘의 Top 3
• 10시 회의 전에 → 전략 자료 준비 (p1)
• 14시~16시 슬롯 → 월간 보고서 초안 (p1)
• 저녁 전 → 팀 메시지 회신 (p2)

📋 기타
• 개발 환경 업데이트 (p3)
• 문서 정리 (p4)

💬 오늘도 집중해서.
```

## 형식 규칙

### Discord markdown 문법
| 형식 | 문법 |
|------|------|
| 굵게 | `**텍스트**` |
| 기울임 | `*텍스트*` |
| 취소선 | `~~텍스트~~` |
| 코드 | `` `코드` `` |

### 섹션별 처리
- **⭐ Top 3**: LLM이 GCal 시간 인접성 기준으로 선정한 오늘 집중 태스크 3개. GCal 일정이 있으면 `시간 컨텍스트 → 태스크 (p우선순위)` 형식.
- **📋 기타**: Top 3에 포함되지 않은 나머지 Todoist 태스크. 없으면 섹션 전체 생략.
- **💬 코멘트**: 1줄 동기부여 또는 하루 요약.

### 글자 수 제한
- 전체 메시지 500자 이내 (모바일 한 화면에 들어오는 밀도)
- 항목이 많으면 요약, 중요도 기준 Top 3 우선

### 톤 지침
- 친근하고 간결한 한국어
- 지나친 격식체나 구어체 지양
- 동기부여 문장은 1줄, 구체적이고 실용적으로

## 예시

```
🌅 오늘의 브리핑 (03/22 토)

⭐ 오늘의 Top 3
• 10시 회의 전에 → 전략 자료 준비 (p1)
• 14시~16시 슬롯 → 월간 보고서 초안 (p1)
• 저녁 전 → 팀 메시지 회신 (p2)

📋 기타
• 개발 환경 업데이트 (p3)

💬 오늘도 하나씩, 차분하게.
```
