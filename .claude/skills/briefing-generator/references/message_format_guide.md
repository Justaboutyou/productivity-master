# Slack 브리핑 메시지 형식 가이드

## 메시지 구조

```
🌅 오늘의 할일 브리핑 (MM/DD 요일)

📌 우선순위 높음
• 할일 항목 1
• 할일 항목 2

📋 일반
• 할일 항목 3
• ~완료된 항목~ (checked: true인 항목)

💬 동기부여 한 마디
```

## 형식 규칙

### Slack mrkdwn 문법
| 형식 | 문법 |
|------|------|
| 굵게 | `*텍스트*` |
| 기울임 | `_텍스트_` |
| 취소선 | `~텍스트~` |
| 코드 | `` `코드` `` |
| 줄바꿈 | `\n` |

### 섹션별 처리
- **top_priorities**: 업무 중요도·긴급도가 높은 항목. `📌 우선순위 높음` 아래 배치.
- **brain_dump**: 메모성 할일. `📋 일반` 아래 배치.
- 완료된 항목 (`checked: true`): 취소선(`~텍스트~`)으로 표시.

### 글자 수 제한
- 전체 메시지 500자 이내 (Slack 미리보기 최적화)
- 항목이 많을 경우 요약하되 중요도 기준으로 우선순위 항목 우선 표시

### 톤 지침
- 친근하고 간결한 한국어
- 지나치게 격식체나 구어체 사용 지양
- 동기부여 문장은 1줄, 구체적이고 실용적인 내용 권장

## 예시

```
🌅 오늘의 할일 브리핑 (03/20 목)

📌 우선순위 높음
• 오전 10시 전략 회의 자료 완성
• 월간 보고서 초안 제출

📋 일반
• 팀 슬랙 메시지 확인 및 회신
• ~Claude Code 설계서 검토~

💬 오늘도 하나씩, 차분하게.
```
